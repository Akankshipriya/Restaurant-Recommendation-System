This folder contains 3 jupiter Notebooks

1. Average_Weighted_Recommendation.ipynb
=======>It is using the geoplaces2.csv and rating_final.csv files to
	show the top Restaurants using average weighted_rating and
	another one showing the top restaurants using popularity and
	weighted_rating both.

2. KNNRecommendation.ipynb
=======>It is using the geoplaces2.csv and rating_final.csv files to
	show the top Restaurants using KNN algorithm.

3. Recommender_Systems_UserID.ipynb
=======>It is using the geoplaces2.csv, rating_final.csv, usercuisine.csv
	and chefmozcuisine.csv files to show the top Restaurants from
	user's past choices and based on popularity.
